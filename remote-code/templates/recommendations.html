The top matches and the recommendations have been computed using a binary vector for memorizing the most common combinations of jobs.

To compute the top matches, the similarities between the skills chosen by the user and each of the vectors are computed and the most similar job patterns.However, because the vectors are binary, the distance between them is represented by the bits that are different, which sometimes, when the user chooses few or unrelated features might cause errors.

The recommendations count the number of patterns where two skills were found together so the number vary more and a more accurate can be done.
